Overview

This Luxe terminal has the developer's Software Development Kit (SDK) installed. The Developer's SDK Terminal contains documentation, tools, and test pages for the Equinox Luxe SDK.

The SDK documentation contains the following sections. Either select the section by clicking on the section links or select the quick links that go directly to the documenation, tool, or test pages

HTTP API

This section of the SDK contains the HTTP API interface specification and a set of web pages to allow a software developer to write and test code that will run externally to the Luxe on the developer's POS / ECR system and communicate to the connected Luxe device through an IP connection using the HTTP API.

  • Luxe HTTP API Specification - Documentation and Test Pages

Payment Engine Integration

This section describes the Equinox Payment Engine and expands on the usage of the HTTP API payment specific commands emv/start and emv/continue

  • Luxe Integration Guide - Documentation and Test Pages
  • Sample ECR Application - Demonstrates EMV Automation
