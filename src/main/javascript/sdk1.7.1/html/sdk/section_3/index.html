Luxe SPDKit - Local Key Injection

SPDKit is a Windows based tool that can be used to inject keys into the Equinox series of terminals including the Luxe
  • SPDKit User Guide
  • Windows SPDKit.exe utility
  • Windows sysWOW64 required DLL's
To install SPDKit.exe on a Windows 10 system:-
  1. Download the Windows SPDKit.exe utility and place it into a convenient location on your computer
  2. Download the Windows sysWOW64 required DLL's .zip file and extract into your c:\Windows\SysWOW64 directory. Note that this may need to be Run as administrator
  3. Open/run a cmd.exe window as Administrator and type the following commands:-
    1. cd \windows\SysWOW64
    2. regsvr32 mscomm32.ocx
